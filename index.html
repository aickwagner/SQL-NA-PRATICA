A função ROW_NUMBER() atribui um número sequencial exclusivo para cada linha em uma partição de resultado. PARTITION BY reinicia a contagem em cada grupo, e ORDER BY define a ordem dentro do grupo.
TABELA: FUNCIONÁRIOS

BrunoVendas7000
DianaVendas5000
RESULTADO DO ROW_NUMBER()
nomedeptosalárionum_linha
SELECT nome, depto, salário,
ROW_NUMBER() OVER (
    PARTITION BY depto
    ORDER BY salário DESC
) AS num_linha
FROM funcionários;
A função RANK() atribui uma classificação a cada linha dentro de uma partição. Diferente de ROW_NUMBER, se houver empates, eles recebem o mesmo valor, e o próximo valor de classificação terá um "salto" correspondente ao número de linhas empatadas.
